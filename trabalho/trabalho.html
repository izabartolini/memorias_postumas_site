Avaliação de Dsesenvolvimento Web e Língua Portuguesa

Essa avaliação tem como finalidade integrar as matérias desenvolvimento web e portugues e literatura. Na primeira matéria é necessário desenvolver páginas web mobile-first e com menu responsivo, conforme aprendido no quarto bimestre, já em literatura á necessário a elaboração de texto do ponto de vista de dois personagens.

Desenvolvimento web

A matéria de desenvolvimento web, ministrada pelo professor Rafael Liberato, visa ensinar a desenvolver páginas web com as linguagens html, css e java script.

Língua Portuguesa e Literatura

A matéria de língua poertugesa e literatura, ministrada pela professora Sandra Aleixo, visa ensinar gramática e estudar sobre gêneros literários clássicos e ler livros dos gêneros estudados.

Trabalho de Desenvolvimento web e Língua portuguesa sobre o livro Memórias Póstumas de Brás Cubas, pelos alunos Izabelly Bartolini e Caio Botelho
